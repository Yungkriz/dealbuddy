# DealBuddy

Static DealBuddy landing page ready for GitHub Pages deployment.

## Deploy to GitHub

1. Create a GitHub repository named `DealBuddy` (or any name you prefer).
2. In this folder, run:
   ```powershell
   git branch -M main
   git remote add origin https://github.com/<your-username>/<your-repo>.git
   git add .
   git commit -m "Initial DealBuddy deployment"
   git push -u origin main
   ```
3. On GitHub, go to **Settings > Pages** and select the `main` branch and `/ (root)` folder.

After a few minutes, your site will be available at `https://<your-username>.github.io/<your-repo>/`.
